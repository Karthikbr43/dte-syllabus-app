<br><br><b>1.Introduction</b>- Linux Architecture- Shell, Kernel, System calls.
 <br>Linux installation- Steps for installing Linux Operating System
 <br>Comparison between Linux and other Operating Systems, Applications of Linux Operating System.

<br><br><b>2.Internal and External commands in Linux.</b>
<br>	• Internal commands- echo, type, etc.
<br>	• External commands- ls, cp, mv, rm, cat, etc
<br>	• Other commands – tput clear, who, cal, date, bc, man, passwd, uname   ( with different options).

<br><br><b>3.Working with files and directories.</b>
	<br>• Know the categories of files.
	<br>• Directory related Commands – pwd, mkdir, rmdir, cd, ls
	<br>• Manipulating Absolute paths and Relative paths using cd command.
	<br>• File related Commands –  cat, cp, mv, rm, comm, cmp, diff, tar, umask, wc

<br><br><b>4.Basic File attributes.</b>
	<br>• Listing seven attributes of a file : ls and its options
	<br>• File Permissions: Absolute and Relative permissions
	<br>• Manipulating File permissions using chmod command
	<br>• Manipulating File Ownership using chown command
	<br>• Manipulating Hardlink and Softlink using ln command

<br><br><b>5.Learn to use vi editor.</b>
	<br>• Three modes of vi editor.
	<br>• Input mode commands.
	<br>• Command mode commands.
	<br>• Ex mode commands.

<br><br><b>6.Simple Filters</b> – head, tail, cut, paste, sort, uniq, tr, pr.
<br><br><b>7.Expressions and search patterns</b>   .(dot operator), *, ^, +, ?, grep, egrep, fgrep
<br><br><b>8.Process Management commands.</b>
	<br>• Process creation, status, Identifying process,  ps -f and its options, 
	<br>• Running process in background, Job control, and Process termination.
	<br>• Changing process priority, scheduling process (Usage of sleep and wait commands)

<br><br><b>9.Introduction to shell programming.</b>
	<br>• Introduction, Uses of shell script, Shell special characters, comments, command separator, escaping, quoting command substitution.
	<br>• Creating shell script, Shell identifiers, Shell variables, Destroying a variable, Positional parameters and command line arguments.
	<br>• Evaluating expressions, Text formatting with echo and tput script termination.
<br><br><b>10.Shell control structures </b>
	<br>• if, case, for, while, relational and logical operators,  
	<br>• Advanced filter – sed and awk.
<br><br><b>11.Linux system administration</b>
      <br>Managing file system, Disk management utilities, mounts, umount, df, du, fdisk, su, useradd etc.     
<br><br><b>12.	Linux Environment </b>
	<br>Introduction, Environment variables, Command prompt system variables, Profiles, files, terminal variable stty command and its options,   Command history, editing Environment variable.
