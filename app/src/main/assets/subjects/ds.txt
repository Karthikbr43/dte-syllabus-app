UNIT I : Pointers and Dynamic Memory allocation				     
<b>Pointers</b> - Concept of pointers, Declaring and initializing pointers, Accessing variables using pointers, Pointer arithmetic, Pointers and arrays, Pointers and character strings, Pointers and functions, Pointer as a function argument, Pointers to  function, Pointers and structures.
<br><b>Dynamic Memory allocation</b> – Introduction, Dynamic memory allocation, Allocating a block of memory:  Malloc, Allocating multiple blocks of memory:  Calloc, Releasing the used space: Free Altering the size of memory:  Realloc 
<br><b>[08 Hrs] [22 Marks] </b>

UNIT II: Files					
Introduction, Defining and opening a file, closing a file, Input / Output operations on files, Error handling during I/O operations, Random Access to files, Command line arguments
<br><b>[06 Hrs] [17 Marks] </b>

UNIT III: Introduction to data structures and Linked lists
<b>Introduction to data structures</b> – Introduction, Characteristics, Types of data structures, data structure operations.
<br><b>Linked lists</b> – Introduction, Basic concept, linked list implementation, Types of linked lists, Circular linked list (no implementation), doubly linked list (no implementation).
<br><b>[10 Hrs] [28 Marks] </b>

UNIT IV: Stack and Queues		
<b>Stack</b> – Introduction, Stacks, Stack operations, stack implementations. 
<br><b>Queues</b> – Introduction, Basic concept, queue operations, queue implementations,  circular queue (no implementation), priority queues (no implementation), double ended queues (no implementation).
<br><b>[10 Hrs] [28 Marks] </b>

UNIT V: Tree				
Introduction, Basic concept, Binary tree, Binary tree representation, Binary tree traversal.
<br><b>[08 Hrs] [22 Marks] </b>

UNIT VI: Sorting, Searching and Application of Data Structures	
<b>Sorting</b> – Introduction, sorting techniques – selection sort, insertion sort, bubble sort, quick sort (no implementation), merge sort (no implementation).
<br><b>Searching</b> – Introduction, Linear search, binary search.
<br><b>Application of data structure</b> – Introduction, Applications of stack, Infix to postfix conversion, Evaluation of a postfix expression, Recursion, factorial, GCD, List application of queues, linked lists and trees.
<br><b>[08 Hrs] [28 Marks] </b>
