UNIT-I: Architecture of 8086 microprocessor
Introduction to Microprocessors, Features, pin functions and internal architecture of 8086. Flag register, Memory segmentation, Segment Registers, Physical address - calculation with examples, Physical memory organization. Interfacing 8086 with memory and I/O devices under minimum mode, Comparison between Minimum mode and Maximum mode configuration 
<br><b>[08 Hrs] [30 Marks] </b>

UNIT-II: 8086 Instruction set
Addressing modes - with example, Role of index and pointer registers, 8086 instruction set-Data transfer, arithmetic, logical, shift and rotate, branching, loop control and string instructions, processor control instructions with simple examples.   	  
<br><b>[10 Hrs] [30 Marks] </b>

UNIT-III: 8086 Programming
ALP program development cycle, development tools, MASM-Assembler directives, structure of assembly program, sample programs -with relevant comments- such as data transfer, code conversion, largest/smallest, sorting, searching, string palindrome and other simple programs. Comparison of procedure and macro
<br><b>[10 Hrs] [25 Marks] </b>

UNIT-IV: Interrupts and Peripheral ICs
<b>Interrupts:</b> Concept, classification–internal and external, maskable and non-maskable, hardware and software. Interrupt Vector Table, interrupt cycle, interrupt service routine, and interrupt priorities. DOS and BIOS routines as interrupt service routines
<br><b>Programmable Peripheral ICs:</b> Functional block diagram, features, various operating modes of IC 8255.  Features and operating modes of 8253, 8259, 8251, & 8257 (only listing), Relevance and features of 8087 co-processor.
<br><b>[09 Hrs] [20 Marks] </b>

UNIT-V: Advanced Microprocessors and Bus Standards
Block diagram of Advanced Microprocessor, Memory Hierarchy, Cache memory, Virtual memory, Paging & segmentation, Pipe lining - Pipe line hazards. Features and comparison of 80286, 80386, 80486, Pentium IV. Concept of core processor.  Bus standards: Need for Bus standards. Features of RS232, Parallel Centronic Bus, SATA Bus, I2C Bus. USB-Structure, operation and features.		
<br><b>[10 Hrs] [30 Marks] </b>

UNIT-VI: Other Advanced Microprocessors
Introduction to Power PC, Features of Power PC601 and AMD Athlon Processor. Features and applications of Super SPARC Processor
<br><b>[05 Hrs] [10 Marks] </b>
